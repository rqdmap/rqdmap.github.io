A C. String Transformation

虽然看上去是一道以字符串为背景的题目，但是其本质是一道图论问题。

(坑)

按照给出的S, T串建出有向图: $E = \{(s[i], t[i])|0 <= i < n\}$。

在A题中，保证图中一定有t[i] > s[i]，那么这张DAG就可以进行拓扑排序。不妨设节点1是第一个入度为0的节点，删去节点1后那些入度成为0的点一定需要花费一次操作代价，而那些不是0的点
